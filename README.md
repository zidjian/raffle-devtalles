## Instalación

1. Clonar el repositorio
2. Copiar el archivo _`.env.examples`_ y renombrarlo a _`.env`_
3. Cambiar las variables de entorno por las deseadas
4. Ejecutar

   ```
   docker-compose up
   ```

5. Ingresar a `http://localhost:3001`
